# Patron-Modelo Vista Controlador
***
Este codigo nos ayudara al ingreso de notas de estudiantes, asi mismo podremos modificar sus notas y varias opciones de un menu.


### Pre-requisitos 📋

Compilador de C++
***
ID Qt Creator
